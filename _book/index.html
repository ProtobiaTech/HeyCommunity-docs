HeyCommunity

HeyCommunity 文档

HeyCommunity 是一个基于 Ionic 开发的开源社交 hybrid app（支持 IOS 和 Android 系统，也支持 WebApp），目前 HeyCommunity 有如下功能和模块:

功能或模块 描述
时间轴 用户可以在时间轴中分享自己的动态和一些有趣的事物
话题 社区话题的交流和讨论
通知 接收社区中与自己有关的信息
用户 基本的用户功能

现在，你可以使用浏览器打开 HeyCommunity WebApp Demo 进行体验和预览。

授权

HeyCommunity 使用 GPLv3 授权，在此此授权协议下你可以:

  • 基于 HeyCommunity 开发自己的社区 app
  • 使用 HeyCommunity 进行商业活动
